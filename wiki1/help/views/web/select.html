select选择器
select选择器是在框架初始化doTree中添加的扩展，包含单选和多选，只需要配置对应参数即可进行初始化使用，配置参数如下： 
val: null,			/* ! 默认值 */
search: false, 		/* ! 是否带search */
multiple: false,	/* ! 是否支持多选 */
split: ',' 			/* ! 多选值，分隔符 */

select属性配置data-val 值为默认值，如果值不存在，则会选择隐藏input的val值，如果值都不存在，或者默认值，在下来选择中不存在，则会选择下来选择的第一个为默认值， 使用时请配置data-do="select"，其他参数根据需要配置,需要的是，带搜索的选择框，需要依赖第三方库pinyin.js，如果配置data-search="true"时，请引入该库
